# Yokohama City Open Data (data-city-yokohama-lg-jp)

Yokohama City Open Data is a municipal government open-data portal for Japan running **CKAN**, exposing a standardized catalog API over ~657 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-city-yokohama-lg-jp/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-city-yokohama-lg-jp-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.9.11

## API
- **Yokohama City Open Data CKAN Action API** — base `https://data.city.yokohama.lg.jp/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://data.city.yokohama.lg.jp)
- Live endpoint verified: `https://data.city.yokohama.lg.jp/api/3/action/status_show` · ~657 datasets
